# Лабораторная работа №1: Установка Linux

## Цель работы

Установить дистрибутив Linux на виртуальную машину 

## Используемое ПО

| Компонент | Версия |
|-----------|--------|
| MacBook | Apple M2 (ARM64) |
| UTM | 4.x |
| Ubuntu | 26.04 LTS Server (ARM64) |

## Ход выполнения

### 1. Установка UTM

Был установлен эмулятор/виртуализатор UTM для запуска ARM64 виртуальных машин на Apple M2.

**Скриншот 1:** Установленная программа UTM

### 2. Загрузка ISO образа

С официального сайта Ubuntu был загружен образ `ubuntu-26.04-live-server-arm64.iso`.  
Архитектура ARM64, так как M2 не поддерживает x86.

**Скриншот 2:** Проблема со входом в линукс

### 3. Создание виртуальной машины в UTM

- **Virtualization Engine:** Apple Virtualization
- **OS:** Linux
- **RAM:** 4 GB
- **CPU:** 4 ядра
- **Disk:** 20 GB (VirtIO)

**Скриншот 3:** Настройка виртуальной машины

### 4. Установка Ubuntu Server

Процесс установки включал:
- Выбор языка (English)
- Настройка сети (DHCP)
- Разметка диска (Use entire disk)
- Создание пользователя:
  - Username: `labuser`
  - Password: `lab2026`
  - Server name: `ubuntulab`

### 5. Настройка SSH

Был установлен OpenSSH Server для удалённого подключения к ВМ из терминала Mac.

**Скриншот 5:** Выбор OpenSSH для подключения к ВМ

![SSH Setup](screenshots/5-ssh-setup.png)

### 6. Первый вход и настройка окружения

После перезагрузки выполнен вход в систему:
login: labuser
password: lab2026

Установлены необходимые инструменты:
sudo apt update && sudo apt upgrade -y
sudo apt install -y build-essential gdb git nano htop

Для удобства работы с лабораторными было настроено подключение к ВМ через терминал Mac:
ssh labuser@192.168.64.2

Результат

Ubuntu Server 26.04 успешно установлен на виртуальную машину, настроен SSH доступ, установлены инструменты для компиляции C++ и написания Bash-скриптов.

Вывод

В ходе работы освоен процесс установки Linux на ARM64 архитектуру через UTM, настройка виртуальной машины, первичная настройка системы и удалённое подключение по SSH.
